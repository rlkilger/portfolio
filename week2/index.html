Javascript: Navice to Ninja Ch. 2 - Programming Basics

C-style syntax - similar to C programming language

Best practice - each statement on it's own line

Reserved words - abstract, await, boolean, break, byte, case, catch, char, class, const, continue, debugger, default, delete, do, double, else, enum, export, extends, false, final, finally, float, for, function, goto, if, implements, import, in instanceof, int, interface, let, long, native, new, null, package, private, protected, public, return, short, static, super, switch, synchronized, this, throw, throws, transient, true, try, typeof, var, volatile, void, while, with, yield

Don't use undefined, NaN, Infinity

Primitive data types - String, Symbol(ES6), Number, Boolean, Undefined, Null

Two variable types(ES6) - const(can't be changed) and let(can be changed) and both have block scope(only exist inside block

const arrays(and other non-prim types) can still be changed using .value

Best practice - always use const when you can

Omitting const and let makes it a global variable

Don't start with _(private) or $(used in jQuery)

Use \ to escape special characters(', ", `, n, r, t, \(\\ escapes \))

Properties can be written name.length or name["length"]

Methods - .toUpperCase(), .nameCharAt(), .indexOf()(if not found, returns -1), .concat()

.isInteger() - returns true or false

Ox = hexidecimal, Oo = octal, 0b = binary

Number methods - careful of decimal(5.0.method(), (5).method(), 5..method(), 5 .method()

Increments - points = 5, points++(returns 5, then adds 1), ++points(adds 1, then returns 6)

Number to big returns Infinity, number too small returns 5e-324 or 0

Number.isFinite() to see if it is a number - returns true or false

Number() to convert string to number, String() to convert number to string(or use .toString()

Convert to int or string using parseInt() or parseFloat() - does not round

NaN = not a number

undefined = no value found

null = no value(behaves like 0)

Falsey values always return false - "", '', ``, 0, -0, NaN, false, null, undefined

!! displays whether true or false, !!"" = false

To find out if NaN, use Number.isNan()

To find out if string is number string, use Number("5") = 5, returns true

Can use greater than and less than on strings, compared alphabetically

Javascript: Navice to Ninja Ch. 3 - Arrays, Logic, and Loops

Arrays area a built-in object

Destructuring arrays - taking a value from an array and assigning it to individual variable(const [x, y] = [1, 2]; x = 1, y = 2)

Swap values - [x, y] = [y, x]; x = 2, y = 1

.length() returns length(does not effect array)

.pop() removes last item and returns it(does effect array)

.shift() removes first item and returns it(does effect array)

.push() adds item to end and returns new length(does effect array)

.unshift() adds item to start and returns new length(does effect array)

.concat() merges arrays(does not effect array)

.join() return array as a string separated by comma, or by value in ()(does not effect array)

.slice(x, y) start at x and end at(excluding) y(does not effect array)

.splice(x, y, "value") starts at x, removes y number of items, inserts "value" in its place(does effect array)

.reverse() reverses array(does effect array)

.sort() sorts alphabetically(for numbers too - [5, 9, 10].sort = [10, 5, 9]

.indexOf() finds value of array(does effect array)

.includes() returns true if value is found(can add index to specify location in array)

Multi Dementional array - const nested = [[1, 2], [a, b]]

Flattened array - const flat = [...x, ...y] = [1, 2, 3, 4]

Create a set - const set = new Set() or new Set([a, b, c]) = Set {a, b, c}

In sets, repeated values only appear once

String in set - new Set("hello") = Set {"h", "e", "l", "l", "o"}

.add() to add to set - new Set().add("hello") = Set {"hello"}

Arrays are unique values and can contain the same values

Different types are different values - {2} != {"2"}

Set values have no index notation

.size() returns number of values

.has() returns true if value is found

.delete("value") returns true if value was found and removed, returns false if value not found

.clear() removes all values from set

Create array from set - const array = [...set] or or const array = Array.from(set)

Memory leak - If you add an array to a set and then delete the original array, set still contains the array values.

To avoid memory leak, use WeakSet() - can only contain non-primitive values and removes dead entries

Create a map - const map = new Map()

.set(1, "a") = Map {1 => "a"} - sets value

.get(1) = "a" - gets value

.has(1) = returns true of value is found

Add multiple values using arrays as parameters - new Map([[1, "a"], [2, "b"]])

.size() finds number of key-value pairs

.delete() returns true if value was found and removed, returns false if value not found

.clear() removes all values from map

Create array from map - const array = Array.from(map) - [[1, "a"], [2, "b"]]

WeakMap() has same differences from map as WeakSet()

Conditional(if statement) - (condition) ? run if true : run if false

Conditional example - (n % 2 === 0) ? "even" : "odd"

While - if breaking at 0, increment can go in condition - let i=5 while(--i) {run code}

Iterator function to iterate through array or set - for(const item of items) {run code}

Iterate through map using .keys(), .values(), or .entries() - for(const key of map.keys()) {run code} or for(const value of map.values()) {run code} or for(const [key, value] of map.entries()) {run code}

Javascript: Navice to Ninja Ch. 4 - Functions

Function declaration - function hello() {concole.log("Hello World!")}

Function expression assigns anonymous function to a variable - const hello = function() {console.log("Hello World!")}

Function() constructor (not recommended) creates global functions- const hello = new Function('console.log("Hello World!");');

If function is assigned to a variable, place parentheses after variable to invoke it as a function

Assign function to another variable - hello = hi; hi() prints "Hello World!"

Functions contain arguments variable that returns parameters like an array (but not an actual array, can't use array methods except length) - function arguments() {return arguments;} arguements(a, b, c) returns {'0':a, '1':b, '2':c}

Rest collects all parameters as array - function rest(...args){for(arg of args){console.log(arg);}} rest(2,4,6,8) logs 2 4 6 8

Default parameter assigned in function - function hello(name='World') {console.log(`Hello ${name}!);} logs "Hello World" if no parameter is given

Default parameters should always come after non-default parameters

Arrow function - const sum = x => x + x;

Multiple parameters - const sum = (x, y) => x + y;

No parameters - const hello = () => console.log("Hello world!");

The return keyword isn't required if the return statement is the only statement in the body of the function

Having multiple lines in body requires curly braces and return statement

Function declaration can be called before it is declared

Callback - function that is passed as an argument to another function

Example of how to use callback - function sing(song,callback) {console.log(`I'm singing along to ${song}.`); callback();}

Callback can also be an anonymous function(only good for one-off tasks) - sing('Let It Go',() => {console.log("I'm standing on my head.");});

Sort numbers with callback function - function numerically(a,b) {return a-b;} [1, 5, 16, 4, 8, 13, 10].sort(numerically);

Use arrow function in .foreach() to iterate through array - colors.forEach((color,index) => console.log(`Color at position ${index} is ${color}`));

.map() also iterates through an array

Examples using .map() - [1,2,3].map(sum); or [1,2,3].map(x => x + x); or ['red','green','blue'].map(color => console.log(`${color.toUpperCase()}`));

Parameters are 1. item in the array, 2. index of item, and 3. the array itself - ['red','green','blue'].map((color, index, array) => console.log(`Element ${index} is ${color}. There are ${array.length} items in total.`));

.reduce() iterates through array and cumulatively combines each result, returns a single value - [1,2,3,4,5].reduce((acc,val) => prev + val);

Used with callback function that takes two parameters - 1. accumulated total and 2. current item

.filter() filters and returns new array

Other methods - reduceRight(), every(), find() and some()

Iterator functions can be chained
