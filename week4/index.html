Javascript: Navice to Ninja Ch. 8 - Forms

document.forms - returns array of every form on the page(works the same as this - document.getElementsByTagname('form')) If you need a specific one, select it using index notation

To reference a specific one, select it using index notation or the form's name attribute - const form = document.forms.search;(to use name, make sure the form's name is unique

Elements - returns an HTML collection of all the elements contained in the form - const [input,button] = form.elements;

Select a specific form control by using it's name attribute - const input = form.searchInput; or bracket notation - const input = form['searchInput'];

Button type submit will manually submit a form

Button type reset will manually reset the fields in a form(not recommended)

Focus event occurs when focused on an element - input.addEventListener('focus', () => alert('focused'), false);

Blur event occurs when user moves focus away from form element - input.addEventListener('blur', () => alert('blurred'), false);

Change event occurs when user moves focus away from form element after changing it - input.addEventListener('change', () => alert('changed'), false);

submit - sends the content of the form to the server to be processed, but you can use JavaScript to intercept the form before itâ€™s sent by adding a submit event listener

value - retrieves value from inside a field - function search(event) {alert(`You Searched for: ${input.value}`); event.preventDefault();}

Link to new form elements - http://www.sitepoint.com/html5-forms-markup

"text" is the default input type - "type='text'" is not needed, but it is good practice to include it

value - set an initial value to a field

checked - set a check or radio as checked by default

type='hidden' - not displayed by the browser, but have a 'value' attribute that can contain information that is submitted with the form(viewable in the page's html

type='file' - used to upload files

It is good practice to use both client-side and server-side validation

Button can be disabled - type='submit' id='submit' disabled

Javascript: Novice to Ninja Ch. 12: Object-Oriented Programming in JavaScript

Can create objects using constructor function - const Dice = function(sides=6){this.sides = sides; this.roll = function() {return Math.floor(this.sides * Math.random() + 1)}}

Create an instance of the object - const redDice = new Dice();

Parentheses are not required when instantiating an object using a constructor function and using the default properties - const redDice = new Dice;
