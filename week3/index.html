Javascript: Navice to Ninja Ch. 5 - Objects

Arrays and functions are objects that are built into the language

Object literal - object that is created directly in the language by wrapping all its properties and methods in curly braces, they make it possible to create an object without having to define a class

If object name is not a valid name for a variable(has space, starts with number, etc), use quotes(but it's preferable to use valid variable names)

Object can be changed even when created using const, and properties don’t always appear in the order they were entered

Create empty object - const object = {}

ES6 way of assigning properties to existing object - variable name becomes the key

Example - const name = 'Iron Man'; const realName = 'Tony Stark'; const ironMan = {name, realName};

Property value can be a Javascript expression - const bewitched = true; const captainBritain = { name: 'Captain Britain', hero: bewitched ? false : true }; returns {name: 'Captain Britain', hero: false}

Check if object has a specific property - 'city' in superman; or superman.city !== undefined; or superman.hasOwnProperty('city');returns true or false

in and !== undefined return true when property is inherited and .hasOwnProperty() does not

Loop through object properties - for(const key in superman) {console.log(key + ":if(superman.hasOwnProperty(key)){console.log(key + ": " + superman[key]);}}

Object.keys() returns an array of the object's keys

Object.values() returns an array of the object's values

Object.entries() returns an array or the object's key-value pairs

Use delete to remove property - delete superman.fly; returns true if property was deleted, false if not found

Can nest objects using curly braces

If variable is assigned to an object that already exists, changes made using either reference will affect the same object

Namespacing - use object literal to create a namespace for groups of related functions

JSON - JavaScript Object Notation

Property names must be double-quoted

Permitted values are double-quoted strings, numbers, true, false, null, arrays and objects

Functions are not permitted values

.parse() - takes a string of data in JSON format and returns a JavaScript object

.stringify() - takies JavaScript object and returns a string of JSON data(methods are ignored)

Can add a space argument that adds new lines between each key-value pair

Math object has built-in properties:

Math.PI - ratio of the circumference and diameter of a circle

Math.SQRT2 - square root of 2

Math.SQRT1_2 - reciprocal of the square root of 2

Math.E - Euler's constant

Math.LN2 - natural logarithm of 2

Math.LN10 - natural logarithm of 10

Math.LOG2E // Log base 2 of Euler's constant

Math.LOG10E // Log base 10 of Euler's constant

Math object has built-in methods:

Math.abs() - returns absolute value

Math.ceil() - rounds up

Math.floor() - rounds down

Math.round() - rounds to nearest integer

Math.trunc() - truncated at decimal point

Math.exp() - raise number in parenthesis to the power of Euler’s constant

Math.pow() - raise first argument to the power of the second argument

Math.sqrt() - returns the positive square root of a number

Math.cbrt() - returns the cube root of numbers

Math.hypot() - returns the square root of the sum of the squares of all its arguments.

Math.max() - returns maximum number from arguments

Math.min() - returns minimum number from arguments

Math.random() - generates number between 0 (inclusive) and 1 (exclusive) - multiply to get random number between 0 and another number

const today = new Date();

today.toString(); returns 'Tue Feb 14 2017 16:35:18 GMT+0000 (GMT)'

Can add argument to Date() to return specific date

January is 0, February is 1, etc.

.getDay() - returns day of the week in number form

.getDate() - returns day of the month

.getMonth() - returns month(starting at 0)

.getFullYear() - returns year

Can use .setDate(), .setMonth(), etc to set date

RegExp Object (regular expression) - search strings for matches to the pattern

Search for string that contain - const pattern = /[a-zA-Z]+ing$/; or const pattern = new RegExp('[a-zA-Z]+ing');(second way can use strings

.test() returns true if pattern is found - pattern.test('joke'); returns false, pattern.test('joking'); returns true, pattern.test('jokingly'); returns false

.exec() - returns an array containing the first match found, or null if there aren’t any matches

pattern.exec('joking'); returns ['joking', index: 0, input: 'joking']
